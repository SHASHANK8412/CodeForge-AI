# 🤝 Contributing to AIForge

Thank you for your interest in contributing to AIForge! We welcome open-source contributions, bug reports, feature requests, and documentation improvements.

## Code Style & Guidelines
- **Backend**: Python 3.11+, PEP8 standards, type hints, docstrings.
- **Frontend**: React 18, ES6+, TailwindCSS, clean functional components.
- **Testing**: Ensure all test suites pass (`pytest` and `python tests/...`).

## Development Process
1. Fork the repository and create a feature branch: `git checkout -b feature/my-new-feature`.
2. Commit your changes with conventional commit messages (`feat: ...`, `fix: ...`, `docs: ...`).
3. Run tests locally and verify functionality.
4. Submit a Pull Request targeting `main`.
