Always use dependency injection.

Thoroughly validate all inputs to ensure data integrity and security, including payloads. Specifically, validate:
- Payload structure
- Data types of each field
- Presence of required fields
- Range or length constraints where applicable
- Sanitize inputs to remove any malicious code

Use SOLID principles.

Include logging to track application behavior and diagnose issues.

Handle exceptions gracefully to maintain system stability.

Implement JWT authentication for all protected user routes to ensure secure access. Ensure that validation is performed at both the API endpoint and within service layers to provide an additional layer of protection.

Explicitly validate payloads in all API endpoints to prevent missing input validation errors. Ensure that payload validation is consistently applied across all endpoints to avoid any potential security vulnerabilities or data integrity issues.
